Day 06 — ChatGPT — Text & Code

System prompts, JSON schemas, small tools and scripts.

Objectives

  • Grasp the principle for today and when to use it.
  • Practice with guided prompts until output is consistent.
  • Ship one artifact (saved to your portfolio folder).

Playbook

  1. Frame: set ROLE, AUDIENCE, OUTCOME.
  2. Constrain: measurable specs (length, format, style, ratios).
  3. Example: give one “gold” example to anchor quality.
  4. Critique: have the model self‑critique → revise.
  5. Export: deliver in required format (Markdown/JSON/PNG/MP4).

Copy‑Paste Prompts

SYSTEM: You are a precise assistant. If constraints are missing, ask clarifying questions.
USER: Create a {deliverable} for {audience} about {topic}.
Constraints: {length / format / tone / acceptance criteria}
Include: a brief rationale and a numbered outline.
Deliver as: {markdown | json | bulleted list}.

Demo

Generate a one‑page brief on “ChatGPT — Text & Code” with KPIs. Convert it into a 5‑slide outline.

Deliverable Checklist

  • Meets constraints exactly.
  • Includes self‑critique → revision step.
  • Exported clearly and saved with date stamp.

Stretch

  • Reduce time‑to‑result by 20% without losing quality.
  • Swap target audience and regenerate.
